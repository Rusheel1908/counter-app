# React Counter App



This is my first React project. I built a simple counter app while learning the basics of React, TypeScript, and React Bootstrap.



## About the Project



The app allows users to increase, decrease, and reset the counter value. I created a reusable button component to understand how components and props work in React.



## Features



- Increment the counter

- Decrement the counter

- Reset the counter

- Reusable button component

- Simple and clean UI



## Built With



- React

- TypeScript

- React Bootstrap

- Vite
